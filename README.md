# 🚀 Jenkins DevOps Pipeline Project

## 📌 Overview
This project demonstrates a CI/CD pipeline using Jenkins to build and push a Docker image.

---

## 🔄 Pipeline Flow
1. Code pushed to GitHub
2. Jenkins pulls code
3. Builds Node.js app
4. Builds Docker image
5. Pushes image to DockerHub

---

## 🧰 Tools Used
- Jenkins
- Docker
- GitHub
- Node.js

---

## 🎯 Goal
To demonstrate Jenkins-based CI/CD automation.
